# ⚖️ Landing Page - Advocacia Profissional

Esta é uma Landing Page moderna e responsiva desenvolvida para um escritório ficticio de advocacia. O projeto foi construído focando em autoridade, confiança e conversão, utilizando as melhores práticas de desenvolvimento web.

---

## 🎨 Design e Identidade Visual

O projeto utiliza uma paleta de cores sóbria para transmitir seriedade:
* **Navy Blue (`#171B24`)**: Base para passar segurança e profundidade.
* **Dourado (`#B99D7C`)**: Detalhes em ouro para simbolizar prestígio e excelência.
* **Branco/Cinza**: Para clareza e legibilidade.

---

## 🛠️ Tecnologias Utilizadas

O site foi construído utilizando tecnologias puras (Vanilla), garantindo leveza e performance:

* **HTML5**
* **CSS3**
* **Font Awesome**
* **Google Fonts**

---

## 📋 Funcionalidades Principal

- [x] **Hero Section**: Com imagem de destaque e chamada para ação (CTA).
- [x] **Diferenciais Estratégicos**: Retângulo flutuante com as principais virtudes do escritório.
- [x] **Áreas de Atuação**: Grade organizada para facilitar a navegação do cliente.
- [x] **Seção Sobre**: Espaço para contar a história e credibilidade do profissional.
- [x] **Contato Integrado**: Localização via Google Maps e botão direto para WhatsApp.
- [x] **Responsividade**: Layout adaptado para celulares, tablets e desktops.
